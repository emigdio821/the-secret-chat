## TheSecretChat

TheSecretChat is just a chat app using Nextjs, Chakra UI and Twilio API. <br />
Follow below instructions if you want to run the project locally üêí.

#### Create an .env file

```
GITHUB_ID=
JWT_SECRET=
NEXTAUTH_URL=
GITHUB_SECRET=
TWILIO_API_KEY=
TWILIO_API_SECRET=
TWILIO_ACCOUNT_SID=
TWILIO_SERVICE_SID=
TWILIO_CHANNEL_ADMIN=
```

#### Install the dependencies
```bash
yarn 
# or 
yarn install
```

#### Run the dev server
```bash
yarn dev
```
<br/>

The project runs on **localhost:3000** by default
