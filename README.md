## The Secret Chat

The Secret Chat is just a simple chat app using Next.js, Tailwind with shadcn/ui, Twilio API and GIPHY API. <br />
Follow below instructions if you want to run the project locally üêí.

#### Create a .env file

```
GITHUB_ID=
JWT_SECRET=
NEXTAUTH_URL=
GITHUB_SECRET=
TWILIO_API_KEY=
TWILIO_API_SECRET=
TWILIO_ACCOUNT_SID=
TWILIO_SERVICE_SID=
NEXT_PUBLIC_GIPHY_API_KEY=
NEXT_PUBLIC_TWILIO_CHANNEL_ADMIN=
```

#### Install the dependencies

```bash
pnpm install
```

#### Run the dev server

```bash
pnpm dev
```

<br/>

The project runs on **localhost:3000** by default
